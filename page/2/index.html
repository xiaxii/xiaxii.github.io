LeetCode #1 Two Sum

Posted on 2020-03-24 | Post modified: 2020-03-24 | In LeetCode

Link: 1. Two Sum

Problem Description

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:
Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

Think out loud

Clarification

  • indices not the position, start from 0
  • exactly one solution any pair that matches requirements, not need a full solution; every test cases should have at least one answer (can be ignored)
  • not use the same element twice the result indeces are different from each other

Assumption
Throw exception when no such an answer

Former Coding Interview Question: Compression and Decompression

Posted on 2020-03-17 | Post modified: 2020-03-17 | In 2019 Google NEXT
  1. Google tech guide — Former Coding Interview Question: Compression and Decompression
  2. Massive Algorithms — LeetCode 394 - Decode String
  3. Massive Algorithms — LeetCode 471 - Encode String with Shortest Length

The Probem is similar to the question like:
Given the string of an arithmatic caculation
eg: {3 x [13 x 21 - (11 + 1) / 3]} / (13 - 2)
Output the result
For the problem above, 2 stacks are needed for decoding (one or operations, another for digital numbers)

Solution of Decoding:

1.使用两个栈，countStack存子串的重复次数，resStack存中间结果
2.读取完一个数字，把数量入countStack栈
3.读取到[时，把n[前面的结果计算出来并入resStack栈
4.读取到]时，取出最近一次n[res]前面的中间计算结果，也就是resStack的栈顶，再append n次res，n为countStack的栈顶，res为最近一次[]中间的子串
5.其他情况，将字符追加到res末尾

The Eagle Project

Posted on 2020-02-09 | Post modified: 2020-09-05 | In Entrepreneurial Project

“The Eagle” Remote Sensing Reconnaissance System

2018 Google - China MoE University-Industry Collaboration Program

This project intends to adopt a light-weight compression deep learning algorithm, based on Google open source artificial intelligence learning system TensorFlow, to propose and implement a set of intelligent processing solutions for mobile remote sensing images (target detection and semantic segmentation) on the mobile artificial intelligence computing platform.

The mobile platform can be carried on the satellite to realize on-orbit real-time remote sensing image information extraction, which provides a basis for subsequent data classification and efficient retrieval, and can filter the target data of interest according to user instructions for downlink transmission, thereby greatly reducing data. The transmission volume, alleviating the bandwidth pressure of the network link, reducing the energy consumption of data transmission, improving the targeting of the downlink data and the timeliness of information acquisition.
